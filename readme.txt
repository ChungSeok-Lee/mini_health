------------------------------------------
총 7가지 등급으로 분류(skinny, cubby, 운동 초기, 운동 상급자, 유튜버들, 피지크 스타, 올림피아) 
1. google_images_download 라이브러리를 사용하여 머신러닝에 필요한 사진들을 다운 받는다.
2. teachable machine에서 각 사진들을 교육시킨다
3. 교육된 모델을 클라우드 상에 업로드하고, 모델의 html코드를 복사해서 가져온다. 
4. html, css코드를 codepen.io에서 가져오고, 약간의 코드를 수정해준다. (웹캠 기능 삭제 & 파일 업로드)
